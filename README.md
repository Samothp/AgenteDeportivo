# Agente Deportivo

Proyecto inicial para crear un agente de análisis de datasets deportivos, enfocado en partidos recientes de fútbol.

## Objetivo

- Ingerir datos de partidos de fútbol
- Extraer métricas clave (goles, posesión, tiros a puerta, faltas, etc.)
- Generar informes claros, visuales y bien estructurados
- Preparar una base para integrar un agente IA que apoye el análisis

## Estructura del proyecto

- `src/` — Código fuente del agente y del análisis de datos
- `data/` — Datasets y archivos CSV de partidos
- `notebooks/` — Exploración interactiva y visualización
- `requirements.txt` — Dependencias del proyecto

## Siguientes pasos

1. Colocar datasets de fútbol en `data/`
2. Ejecutar `pip install -r requirements.txt`
3. Usar `src/agent.py` y `src/data_loader.py` para cargar y analizar datos
4. Crear notebooks en `notebooks/` para visualizaciones y reportes
